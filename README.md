# Transcendence

### Install
```zsh
git clone git@github.com:0x21x/ft_transcendence.git
```

### Run

### # Development
```zsh
docker-compose --file docker-compose-dev.yml up --build --watch
```
### # Production
```zsh
docker-compose up --build
```

### Stop
```zsh
docker-compose down
```
